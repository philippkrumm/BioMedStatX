# Excel Templates and Sample Files

This file points to the sample Excel template you can use to try the HowTo and ANOVA examples.

- Repository template: `docs/StatisticalAnalyzer_Excel_Template.xlsx` — this file is the recommended template for testing and examples.

- Precompiled builds: some release bundles (Windows/macOS) include a template named `BioMedStatX_excel_template.xlsx`. If you downloaded a release, you may find that copy in the release assets or the application package.

Which one to use?

- Use `docs/StatisticalAnalyzer_Excel_Template.xlsx` to follow the documentation examples in `docs/HowTo.md` and `docs/ADVANCED_ANOVA_GUIDE.md`.
How to open it:

- In Excel (Windows/macOS): double-click the `.xlsx` file.
- In LibreOffice (Linux): `libreoffice --calc docs/StatisticalAnalyzer_Excel_Template.xlsx`
